A simple and efficient task management web application designed to help users organize their daily activities.

The app was built using PHP with the MeekroDB library for database interaction, providing a lightweight and straightforward solution for managing tasks. It includes features such as task categorization, priority settings, and a user-friendly interface. The backend ensures data integrity and efficient queries, while the frontend delivers an intuitive experience.

Technologies Used:

  • Language: PHP
  • Database Library: MeekroDB
  • Database:MySQL
  • Frontend: HTML, CSS, JavaScript
