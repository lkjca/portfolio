VOYAGIO Travel Planner

A Travel Planner web app built using React that helps users plan and organize their trips efficiently.

See Demo Frontend Repository Backend Repository

VOYAGIO Travel Planner is an innovative web application designed to streamline trip planning and enhance user experiences through advanced technology. Built with React.js for the frontend and Node.js with Express for the backend, the app leverages the Google Maps API to enable users to search for destinations, routes, and nearby attractions. It also recommends related points of interest, such as restaurants and landmarks, based on user preferences. The app integrates AWS S3 for efficient data management, allowing users to upload photos to document their travel experiences. With a seamless interface and powerful features, VOYAGIO simplifies trip organization and creates personalized travel plans, making every journey memorable and hassle-free.

Technologies Used:

  • Frontend: React.js
  • Backend: Node.js with Express
  • Database: DynamoDB
  • Mapping: Google Maps API
  • Authentication: JSON Web Tokens (JWT)
  • Hosting: AWS or Heroku
