LINKEDIN CLONE

A dynamic and feature-rich web application designed to mimic the core functionalities of the LinkedIn platform.

See Repository

The LinkedIn Clone is a dynamic and feature-rich web application designed to mimic the core functionalities of the LinkedIn platform. It enables users to create profiles, connect with professionals, share updates, and engage in networking activities. The application leverages AWS S3 for secure and scalable storage of user-uploaded files, such as profile pictures and posts, and AWS RDS for a robust and reliable relational database solution. Built with Spring Boot, the project integrates Spring Security to enforce authentication and authorization, ensuring data privacy and secure access. On the frontend, it employs Thymeleaf as a server-side template engine for rendering dynamic content, complemented by Quill.js, a rich-text editor that enhances user interaction for posting updates and creating professional profiles.

Technologies Used:

  • Frontend: Bootstrap
  • Backend: Spring Boot
  • Database and Storage: AWS RDS, AWS S3
  • Rich Text Editing: Quill.js
  • Authentication: Spring Security
  • Hosting: Heroku
